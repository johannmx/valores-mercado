# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do not disclose it publicly. Instead, please report it by sending an email to [dev@johatech.ar](mailto:dev@johatech.ar).

We take security seriously and will investigate all reports promptly.

## Supported Versions

Currently, only the latest version of the `main` branch is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| < 1.0.0 | :x:                |

## Security Headers and Best Practices

This project aims to follow web security best practices, including:
- Proper security headers (Content-Security-Policy, etc.)
- Regular dependency updates via Dependabot
- Minimal attack surface in Docker configurations
